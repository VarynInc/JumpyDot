Changes Made to WordPress

Before you upgrade you should do three things:

  1. Make a backup of the entire /blog folder in case you need to reference something overwritten in an upgrade. Also note the entire website is versioned with git so you can rely on that if your working copy is up to date.
  2. Make a backup of the entire database. There is a script on the server to do this.
