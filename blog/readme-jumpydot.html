JumpyDot WordPress Notes

We made several changes to the WordPress template, this document will attempt to indicate the changes that were made. Next time we upgrade hopefully we will not lose anything

Changes Made to WordPress

Before you upgrade you should do three things:

  1. Make a backup of the entire /blog folder in case you need to reference something overwritten in an upgrade.
  2. Make a backup of the entire database.

wp-config.php

Note the reference to /procs/common.php this gets the database connection info.

Setup the database connection info and the WP authentication keys

wp-content/themes

Added the ViewPoint theme from Theme Forest

Edited the following theme files. Do a diff on these files if you update so you can reapply the changes.

  • header.php
  • css/base.css
  • css/layout.css
  • images/* (updated to JumpyDot images)

wp-content/uploads

Grab all the uploaded media files from here that will be referenced in Posts and Pages.
